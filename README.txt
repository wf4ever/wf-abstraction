wf-Abstraction WP2-Wf4Ever T2.4

This project is java application to create a trie strcuture including the relative frequencies of executed resources of a set of workflows. The app. (wfAbstraction.jar) accesses to the repository at http://repos-wf4ever.isoco.com/sparql to retrieve the stored provenance of workflow results and creates an output XML file with the relative frecuencies by level of the ordered executed resources wfTrie.xml. 

Installation instructions: the whole project can be compiled using Maven. The description of the project is in its pom.xml file.
